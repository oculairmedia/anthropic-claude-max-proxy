# LLMux Documentation

This folder contains focused guides that expand on the quick overview in the top-level README:

- `authentication.md` – Provider OAuth flows, token storage, long-lived token helper.
- `headless.md` – Running the proxy in headless mode for CI/CD or containers.
- `configuration.md` – Environment variables, `.env` usage, CLI flag reference.
- `models.md` – Available model identifiers and capability notes.
- `custom-models.md` – Adding OpenAI-compatible providers via `models.json`.
- `client-usage.md` – Sample code snippets and API usage across SDKs.
- `reasoning.md` – Enabling and tuning reasoning/thinking support.
